Projetos Desenvolvidos

Abaixo está os projetos desenvolvidos no Curso Técnico de ADS :)

CodeKidsLab

Este projeto consiste no desenvolvimento de uma plataforma de estudos focada
em conteúdos de programação. Através dela, instrutores podem publicar atividades práticas,
materiais didáticos e vídeo-aulas sobre diversas linguagens e tecnologias. Os alunos têm acesso
completo a todo esse conteúdo, podendo acompanhar as aulas, resolver desafios de código e aprofundar
seus conhecimentos de forma estruturada e acessível.

Estilo e Charme

Este projeto é um site desenvolvido para uma loja de roupas que, além de
exibir e vender produtos, oferece funcionalidades inteligentes para melhorar
a experiência do cliente. A plataforma permite calcular o IMC (Índice de Massa Corporal)
do usuário e, com base no resultado, gera indicações personalizadas de vestuário, considerando
o biotipo. O sistema também possibilita o cadastro de clientes e a consulta de seus dados, facilitando o
gerenciamento e o relacionamento com o público. A proposta combina e-commerce com tecnologia
para oferecer recomendações mais assertivas e uma navegação mais personalizada.

CrVe

Sistema completo de gerenciamento de serviços que inclui controle de estoque
(com adição e atualização de produtos), cadastro e gerenciamento de funcionários,
além da organização dos serviços prestados. A plataforma permite acompanhar operações,
produtos utilizados, equipe envolvida e oferece suporte à tomada de decisões com uma interface
prática e eficiente.

Qfome

Plataforma de e-commerce para delivery com funcionalidades de cadastro
de empresas, realização de pedidos, busca por categorias ou lojas, e
um diferencial de integração com uma API de mapas ucraniana que
localiza os estabelecimentos por latitude e longitude.
